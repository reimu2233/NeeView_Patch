# WebDAV 115 网盘配置（NeeView 魔改版）

> 这个 WebDAV 功能我自己专门为看 115 漫画设计的，因此只针对 115 网盘做了优化。

- ① 查看 [alist 官方文档](https://alistgo.com/zh/guide/drivers/115_open.html) 给 alist 添加 `115open` 存储。  
  **注意**：添加存储时不要使用 alist 的 “115 网盘/分享”，这种方式非常容易触发风控。
- ② 打开 NeeView 魔改版的设置界面 → **WebDAV**，填写相应信息。
- ③ 地址填写示例：`http://127.0.0.1:5244`  
  WebDAV 根路径示例：`/dav/115网盘`  
  （具体路径请根据你自己 alist 的挂载路径填写；也可以直接用 `/dav` 挂载整个 alist，但对 115 以外的存储兼容性未知）
- ⑤ 如果挂载时出现 404 或 505 错误，很可能是 115 的风控导致的，稍等一段时间通常可以恢复。  
  也可以在 **设置 → WebDAV → 操作 → 设置 → 高级选项** 里调节 API 获取速度。  
  我目前设置的参数是 `4` 和 `3`，使用这套值几乎再没遇到过风控。
